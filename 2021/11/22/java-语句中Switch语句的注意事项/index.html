java 语句中Switch语句的注意事项

2021-11-22 415 字
java学习

Switch语句的注意事项

1.表达式数据类型，应和case后的常量类型一致，或者是可以自动转成可以相互比较类型，比如输入的是字符，而常量是int

char类型是可以转换为String类型的，例如下面语句为正确：

char c = 'a';
		switch(day){
			//在java中，只要有返回值，就是一个表达式
			case 'a' :
			System.out.println("星期一");
			break;
			//如果没有break语句，则直接执行case 20 :
			//语句里面的代码块
			case 20 :
			System.out.println("星期二");
			break;//退出switch语句，但不是退出程序
			default:
			System.out.println("输入错误");
			break;//default后面的break可以省略
        }
//char day = 'a';//char类型的定义
//String day = "ab";//字符串类型的定义

2.Switch(表达式)中表达式的返回值必须是：(byte,short,int,char,enum[枚举],String)

  1. case子句中的值必须是常量或者是常量表达式,而不能是变量

    char c = 'a';
    char c1 = 'b';
    		switch(day){
    			//在java中，只要有返回值，就是一个表达式
    			case 'a' :
    			System.out.println("星期一");
    			break;
    			case c1 ://c1为变量，所以case c1 :错误
    			System.out.println("星期二");
    			break;
                case 'a'+1 :
    			System.out.println("星期三");
    			break;

  2. default子句是可选的， 当没有匹配的case时，执行default,如果没有default语句，又没有匹配任何常量，则没有输出；

  3. break语句用来在执行完一 个case分支后使程序跳出switch语句块; 如果没有写break,程序会顺序执行到switch结尾，除非遇到break语句
