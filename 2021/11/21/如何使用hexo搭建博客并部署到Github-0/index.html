3.GitHub的注册

5.利用hexo创建博客并上传GitHub

(1)在sources文件下的_posts下创建.md文件，利用Typora编写博客

(2)打开Git

​ hexo clean;

​ hexo g;//生成静态网页

​ hexo s;//本地浏览

​ hexo d;//部署到GitHub，上传的密码是SSH key
